# ShiftChange 🏥

A modern nursing platform for shift management, job opportunities, continuing education, and professional community.

## Features

✅ **Job Board** - Healthcare job listings with filtering and search  
✅ **Shift Marketplace** - Open shifts, swap requests, and shift matching  
✅ **CEU Tracker** - Track continuing education credits with working provider links  
✅ **Nurse Profiles** - Complete nurse profiles with specialties and experience  
✅ **Social Feed** - Post updates, ask clinical Q&A, build community  
✅ **Mentoring** - Connect with experienced nurses for career guidance  
✅ **Mobile Responsive** - Works perfectly on all devices  

## Tech Stack

- **Frontend:** Next.js 14, React 18, Tailwind CSS
- **Backend:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Deployment:** Vercel
